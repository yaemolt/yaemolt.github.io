使用 GitHub

将代码推送到 GitHub

如果想在 GitHub 上共享本地仓库，首先将其连接到远程 Git 仓库：

  • git remote add origin <url>：添加一个新的远程仓库。
  • git branch -M main：将当前分支重命名为 main。
  • git push -u origin main：将本地仓库推送到 GitHub，并设置 origin/main 作为默认的上游分支。
